# Silverblue deploy repo for SZH

Image Fedora Silverblue dérivée pour une flotte de **ThinkPad**, construite et
publiée via GitHub Actions selon le modèle *image mode / bootc* (config versionnée,
déploiement atomique, rollback en un redémarrage).

## En cas d'interet
Notre documentation est privée, en cas d'intérêt nous transmettons volontiers les détails de notre configuration à toutes les personnes interessées.
